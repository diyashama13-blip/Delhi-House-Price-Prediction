# 🏠 Delhi House Price Prediction

## 📌 Overview

This project focuses on predicting house prices in Delhi using Machine Learning techniques.
It includes data preprocessing, exploratory data analysis (EDA), model building, and performance evaluation.

---

## ⚙️ Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib & Seaborn
* Scikit-learn

---

## 📊 Features of Project

* Data Cleaning & Preprocessing
* Exploratory Data Analysis (EDA)
* Feature Engineering
* Model Training (Linear Regression, Random Forest, XGBoost)
* Performance Evaluation

---

## 📈 Results

The model successfully predicts house prices with good accuracy and provides insights into key factors affecting pricing.
